# Instructions pour créer les icônes PWA

## Icônes nécessaires

Créez les icônes suivantes dans le dossier `/icons/` :

### Icônes principales (PWA)
- `icon-72x72.png` (72x72px)
- `icon-96x96.png` (96x96px)
- `icon-128x128.png` (128x128px)
- `icon-144x144.png` (144x144px)
- `icon-152x152.png` (152x152px)
- `icon-192x192.png` (192x192px) - **Maskable**
- `icon-384x384.png` (384x384px)
- `icon-512x512.png` (512x512px) - **Maskable**

### Icônes Apple
- `apple-touch-icon.png` (180x180px)

### Favicons
- `favicon-16x16.png` (16x16px)
- `favicon-32x32.png` (32x32px)
- `safari-pinned-tab.svg` (Monochrome SVG)

### Raccourcis
- `shortcut-booster.png` (96x96px)
- `shortcut-collection.png` (96x96px)

### Screenshots
- `screenshot-1.png` (1280x720px - Desktop)
- `screenshot-2.png` (750x1334px - Mobile)

### Image Open Graph
- `og-image.png` (1200x630px) - À la racine du site

## Recommandations

1. **Design**: Utilisez un logo Pokéball ou carte Pokémon stylisée
2. **Couleurs**: Fond bleu (#3b4cca) avec accent jaune (#ffde00)
3. **Maskable icons**: Ajoutez 20% de safe zone autour du design principal
4. **Format**: PNG avec transparence pour les icônes
5. **Optimisation**: Compressez les images pour de meilleures performances

## Génération rapide

Vous pouvez utiliser des outils en ligne comme :
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator

Uploadez une image 512x512px et générez automatiquement toutes les tailles.
